# Laboratory Activity #3: Working with Sensors

This activity introduces sensors. It reads data from a Thermistor (temperature) and a Photoresistor (light). It uses logic to determine if a "Fire" is present (High Temp + High Light) and triggers an alert (LED/Buzzer) if specific thresholds are met.
